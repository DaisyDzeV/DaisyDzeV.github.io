Progressive Web Application (PWA)

Some characteristics of Progressive Web Applications or PWAs are that they are fully responsive, allow for push notifications, and the interface is consistent with the mobile app interface. Some specific characteristics discussed in the lecture were HTTP caching, web push notifications and web manifest. They are important because they ensure that users get an improved and engaging experience while using apps.

The characteristics that stood out the most is that PWAs can be accessed offline. This can greatly increase the user experience while interacting with a web app. PWAs can also be used on browsers with device features and styles. This characteristic is important because it creates consistency and familiarity for users, which is important for user experiences. These two things contribute to what makes PWAs successful and stand out from native apps.
