Progressive Web Application (PWA)

Some characteristics of Progressive Web Applications or PWAs are fully responsive, allow for push notifications, and the interface is consistent with mobile app interface. Some specific characteristics discussed in the lecture were HTTP caching, web push notifications and web manifest. They are important becuase they ensure that users get a improved and engaging experience while using products.

The characteristics that stood out were that the can be accessed offline and they can be used on browesers with device features and styles. These two things are what seperate PWAs from native apps.

Create service-worker.js file to make all PWA work offline
